Bienvendo a nuestra documentacion

¡Bienvenido a la biblioteca Real Madrid CSS/HTML! Eleva tus proyectos web al siguiente nivel con nuestra colección exclusiva de componentes diseñados para reflejar el espíritu y la grandeza del club más emblemático del fútbol.

Desde botones que emulan la pasión de la afición merengue hasta tarjetas que despliegan la gloria de sus triunfos históricos, nuestra librería ofrece una variedad de elementos cuidadosamente diseñados para adaptarse a cualquier proyecto relacionado con el Real Madrid.

Explora nuestra selección de formularios que capturan la elegancia y la determinación de los jugadores del Real Madrid en el campo. Sumérgete en nuestras alertas dinámicas que recuerdan la intensidad de los momentos clave en la historia del club.

Además, nuestra barra de navegación te llevará a un viaje fluido por tu sitio web, mientras que los distintivos (badges) te permitirán resaltar los logros y reconocimientos de tu plataforma al estilo madridista.

¡Con contenedores diseñados para brindar la mejor experiencia de usuario y una cobertura total sobre CSS y HTML, nuestra librería Real Madrid es la elección perfecta para que tu sitio web luzca como un verdadero campeón!

Width

La propiedad CSS width establece el ancho de un elemento.

En nuestra libreria, establecemos el width en base a porcentajes que van desde 10% hasta 100%, ademas, se agrega un valor especial, el cual es el valor auto.

Como caracteristica de nuestra libreria, las clases de width pueden funcionar distinto dependiendo la resolucion de la pantalla, por ejemplo, existe la clase "wd-50" que establece el ancho del elemento en un 50%, esta clase funciona para todas las resoluciones, pero ademas existen otras tres clases que establecen el ancho en 50% dependiendo de la resolucion, en el caso de la clase "width-50" existe "sm-wd-50", la cual establece el ancho del elemento en un 50% si la resolucion es menor a 500px, despues, tenemos "md-wd-50", la cual establece el ancho del elemento en un 50% si la resolucion es mayor a 501px pero menor a 1023px y por ultimo, tenemos "bg-wd-50", la cual establece el ancho del elemento en un 50% si la resolucion es mayor a 1024px.

Flexbox

Flexbox es un método de diseño de página unidimensional para compaginar elementos en filas o columnas. Los elementos de contenido se ensanchan para rellenar el espacio adicional y se encogen para caber en espacios más pequeños.

En nuestra libreria, establecemos todas las propiedades de Flexbox.

Como caracteristica de nuestra libreria, las clases de Flexbox pueden funcionar distinto dependiendo la resolucion de la pantalla, por ejemplo, existe la clase "flex-d-col" que establece que los elementos se colocan uno debajo del otro, esta clase funciona para todas las resoluciones, pero ademas existen otras tres clases que establecen propiedades de flex dependiendo de la resolucion, en el caso de la clase "flex-d-col" existe "sm-flex-d-col", la cual establece que los objetos se colocaran uno debajo del otro si la resolucion es menor a 500px, despues, tenemos "md-flex-d-row", la cual establece que los elementos se colocarán uno al a par de tro si la resolucion es mayor a 501px pero menor a 1023px y por ultimo, tambien existen las clase si la resolucion es mayor a 1024px.

Grid

El Grid layout permite a un autor alinear elementos en columnas y filas. Sin embargo, con CSS grid son posibles muchos más diseños y de forma más sencilla que con las tablas. Por ejemplo, los elementos secundarios de un contenedor de cuadrícula podrían posicionarse para que se solapen y se superpongan, de forma similar a los elementos posicionados en CSS.

Nuestra libreria implementa la mayoria de propiedades de Grid.

Como caracteristica de nuestra libreria, las clases de Grid pueden funcionar distinto dependiendo la resolucion de la pantalla, por ejemplo, existe la clase "gris" que establece el display: grid, esta clase funciona para todas las resoluciones, pero ademas existen otras tres clases que establecen display: grid dependiendo de la resolucion, en el caso de la clase "grid" existe "sm-grid", la cual establece display: grid si la resolucion es menor a 500px, despues, tenemos "md-grid", la cual establece display: grid si la resolucion es mayor a 501px pero menor a 1023px y por ultimo, tambien existen las clase si la resolucion es mayor a 1024px.

Formularios

En el formulario, los campos de entrada están distribuidos en filas y columnas utilizando Flexbox, lo que permite una disposición flexible y adaptable a distintos dispositivos. Cada campo de entrada está etiquetado de manera clara y concisa para indicar qué tipo de información se espera del usuario.

Row

Cuando se aplica la clase .row a un contenedor HTML, se le está dando la instrucción de que los elementos dentro de este contenedor se alineen en una fila horizontal. En el contexto de este código, se utiliza para agrupar elementos que deben mostrarse en una fila, como etiquetas de texto y campos de entrada. 

                        
                            
<div class = "row"> </div>

Column

La clase .column se utiliza para definir un contenedor flexible dentro de un diseño de columnas en CSS. Cuando se aplica esta clase a un elemento HTML, se le está indicando que se comporte como una columna dentro de un diseño flexible, lo que significa que los elementos dentro de este contenedor se alinearán en una columna vertical. 

                        
                            
<div class = "column">

  //Esta seria una manera de usar

</div>

                            
<div class = "row">
      <div class = "column">

            //Esta seria una manera de usar

      </div>
</div>
